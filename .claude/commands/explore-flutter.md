# Explore Flutter Reference

Guide to exploring the reference Flutter implementations.

Key patterns from `../soliplex_flutter/`:

- `lib/client/` - API client and AG-UI integration
- `lib/providers/` - Riverpod state management
- `lib/features/` - UI feature modules
- `lib/app_shell.dart` - 3-column responsive layout

Patterns: Riverpod, http package (not Dio), AG-UI SDK, Material Design 3.
